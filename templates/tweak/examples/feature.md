# Exemplos de demandas para `--template tweak`

Aceitas:

- Mude a cor do botão Salvar para azul.
- Troque o texto “Enviar” por “Publicar”.
- Reduza o espaçamento entre o título e o subtítulo para 8px.
- Corrija o ícone de fechar que fica desalinhado no modal.

Devem usar `--template feature`:

- Adicione um novo fluxo de checkout.
- Crie um endpoint e persista os dados no banco.
- Troque a biblioteca de componentes.
- Refatore a autenticação e as permissões.
